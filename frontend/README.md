# Frontend

Next.js application for ThoKan Cloud.

## Setup

```bash
cd frontend
cp .env.example .env.local
npm install
npm run dev
```
## Environment

NEXT_PUBLIC_API_BASE_URL=http://localhost:8000/api/v1

# Optional: native iOS/Android API URL override (use your Mac LAN IP for physical devices)
# Example: http://192.168.1.42:8000/api/v1
NEXT_PUBLIC_NATIVE_API_BASE_URL=

App runs on `http://localhost:3000`.
